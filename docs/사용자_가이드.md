# 사용자 가이드

## 목적

이 앱은 코일 실험 데이터를 업로드해 파형을 검수하고, Quick LUT 운용을 통해 전압 파형과 주파수에 따른 자기장 파형을 확인하는 로컬 도구입니다.

## 앱 실행

전체 앱:

```powershell
.\launch_field_analysis_latest_local.cmd
```

Quick LUT 앱:

```powershell
.\launch_quick_lut_local.cmd
```

차이:

- `launch_field_analysis_latest_local.cmd`: Data Import, Raw Waveforms, Field Diagnostics, Validation/Retune, Export까지 포함한 전체 화면입니다.
- `launch_quick_lut_local.cmd`: Quick LUT 운용과 관련 검수 화면을 빠르게 확인하는 화면입니다.

화면이 변경되지 않은 것처럼 보이면 기존 Streamlit 프로세스를 종료한 뒤 launcher를 다시 실행하십시오.

## 기본 사용 흐름

1. CSV 또는 Excel 데이터를 업로드합니다.
2. Data Import에서 매핑과 메타데이터가 맞는지 확인합니다.
3. Raw Waveforms에서 raw/corrected 파형을 직접 검수합니다.
4. Quick LUT에서 field-only / rounded-triangle / 100pp fixed 운용 상태를 확인합니다.
5. 필요한 경우 export 또는 PR 검토용 산출물을 저장합니다.

## Raw Waveforms 검수 포인트

- `테스트 선택`: 사람이 읽을 수 있는 metadata label입니다.
- `Internal ID`: 내부 식별자입니다. 파일 hash나 test id는 여기서만 확인합니다.
- `Source file`: 실제 입력 파일명입니다.
- `raw/corrected`: raw는 파싱 직후 데이터, corrected는 전처리 후 데이터입니다.
- finite-cycle marker: nonzero start/end, target active end, zero/tail 구간을 확인합니다.
- anomaly quick checks: pp, rms, max adjacent jump, spike/clipping/flatline/duration mismatch 의심을 빠르게 봅니다.
- `비교 기준 테스트 (선택)`: 여러 파형을 비교할 때 쓰는 optional baseline입니다. 단일 데이터 검수 시에는 `없음`으로 두면 됩니다.

## 주의

Raw Waveforms는 데이터 검수 화면입니다. 여기서 표시되는 anomaly flag는 모델링/solver를 바꾸지 않으며, 사람이 재측정 또는 제외 여부를 판단하기 위한 참고 정보입니다.
