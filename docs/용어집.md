# 용어집

## Physical Target

사용자가 만들고 싶은 실제 자기장 파형입니다. 현재 target은 rounded triangle / 100pp fixed입니다.

## Support Reference

실험 데이터에서 가져온 support 기준 파형입니다. Physical Target이 아니라 비교/보간 reference입니다.

## Predicted Output

추천 command를 넣었을 때 예상되는 자기장 출력입니다.

## finite empirical route

finite-cycle transient support data를 사용해 finite-cycle 결과를 계산하는 경로입니다. exact support가 없으면 fallback 또는 unavailable 상태가 될 수 있습니다.

## support family override

요청한 support/input waveform family와 실제 선택된 support family가 달라지는 경우입니다. 더 안정적인 support가 있으면 override가 표시될 수 있습니다.

## Raw Waveforms

raw/corrected 파형을 직접 보고 데이터 품질을 검수하는 화면입니다. 모델링 전 데이터 audit의 중심 화면입니다.

## internal id

앱 내부 test 식별자입니다. 사람이 데이터를 찾을 때는 metadata label과 source file을 우선 사용합니다.

## source file

업로드된 원본 파일명입니다. 새 데이터셋은 `continuous_{waveform}_{freq}Hz.csv`, `finite_{waveform}_{freq}Hz_{cycle}cycle.csv` 규칙을 권장합니다.

## quarantine

이상 데이터가 의심되어 모델링 support에서 제외하거나 별도로 보관하는 상태입니다. spike, clipping, truncation, wrong cycle, sign 오류 등이 후보입니다.

## retest required

재측정이 필요한 상태입니다. Raw Waveforms에서 source file, marker, anomaly quick checks를 확인한 뒤 판단합니다.

## target_end

사용자가 원하는 physical target 종료 시점입니다.

## command_end

target을 만들기 위해 command를 끝내는 시점입니다.

## predicted_settle_end

predicted field가 실제로 안정화되는 시점입니다.
